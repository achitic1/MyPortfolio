Project: Lambda Calculus Interpreter
Description: Implemented a lexer, parser, and evaluator to be used for a Lambda Calculus Interpreter as well as an English to Lambda Calculus complier. The project was restrained to only using functional OCaml no imperative features were permitted. The lexer made use of regular expressions, a recursive descent parser was implemented, functions to reduce lambda expressions were also implemented. This project supplied me with a deeper understanding about the workings of compliers and interpreters.

Technologies Used:

  • OCaml
  • Bash Shell
  • Git
